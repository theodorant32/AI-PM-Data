# Weather Forecasting with XGBoost

A time-series forecasting project that uses XGBoost to predict temperature trends from historical weather data.

## Dataset

Global Weather Repository dataset from Kaggle containing historical weather observations.

## Project Structure

- `data/` - Raw and processed data
- `notebooks/` - Jupyter notebooks for each analysis stage
- `outputs/` - Generated plots and predictions
- `scripts/` - Utility scripts

## Getting Started

```bash
pip install -r requirements.txt
python scripts/generate_sample_data.py
jupyter notebook
```

Run notebooks in order: 01 → 02 → 03 → 04
